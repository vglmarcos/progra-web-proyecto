Bienvenidos

En esta página web se encontraran temas diversos de ciencias computacionales, como lenguajes más comunes para el desarrollo web; programación básica: estructuras, recursividad, arreglos y programación orientada a objetos; y algoritmos computacionales de búsqueda y ordenamiento.
Cada uno de estos temas sirve como apoyo para los estudiantes de ciencias computacionales para documentarse correctamente sobre temas básicos para cualquier programador, además contiene videos de prueba y ejemplos prácticos de algunos temas.
